about=A combined RORB toolbox with three toolbar icons:
    1. Create RORB Layers — create centroid, confluence and reach layer templates
    2. Build RORB .catg — auto-name layers, check links, build RORB/URBS control vectors
    3. RORB Results Viewer — browse .out files, plot hydrographs, export critical events

    Bundles pyromb>=0.3 for the build tools — no separate install needed.

changelog=
v2.17: Run RORB: fix "No runs" error when using ARR Data Hub IFD depth CSVs
    whose AEP headers are formatted as "1% AEP" instead of "1%" — the
    normaliser now strips the trailing " AEP" suffix before matching.
v2.16: Results Viewer: Export tab preview table now supports right-click to
    open the .out or .stm file for any listed event, or to open the
    containing folder — matching the Critical Events tab behaviour.
v2.15: Results Viewer: add "Export Temporal Patterns (_tp.csv)" button — exports
    the temporal pattern as % of burst depth per time step for each selected event.
v2.14: Run RORB: fix ISA group parsing — Parameters table now shows the correct
    7.2 print nodes and outlet from the .catg (was showing only "outlet").
    Add Refresh button in Parameter configuration to re-read ISA groups from
    the current .catg without re-browsing.
v2.13: Run RORB: fix Parameters table in non-lumped (vary) mode — rows now show
    ISA groups (matching RORBwin's Parameter Specification dialog) instead of
    individual sub-areas.
v2.12: Run RORB: add "Suggest Kc" helper dialog — choose a regional equation
    (ARR Book 7 area-based or Dav-based Pearse et al.) or the RORB Manual
    default (2.2 × √A) to compute a starting kc for the selected ISA group,
    then adopt it into the Parameters table with one click.
v2.11: Results Viewer: fix Critical Events hydrograph x-axis clipping to 2 hr when
    the selected event has zero flow — now shows the full storm duration so rainfall
    bars are visible.
v2.10: Build .catg: fix sub-catchment areas being wrong (stuck at the default
    value from the first polygon) for some polygons — replace the fragile
    polygon-id-field lookup with a direct centroid→polygon spatial join; also
    fix total-area display double-counting when two centroid points share the
    same polygon.
v2.9: Run RORB: new Calc. Order tab — browse a .catg file to see print nodes,
    their names and average distances in calculation order. Build .catg: new
    "Rewrite input files" output mode overwrites original shapefiles in place
    (with retry for Windows file-lock). pyromb fix: handle unvisited traversal
    states; prevent duplicate node IDs in the graphics block.
v2.8: Build .catg: fix print-node location names being dropped (and stray
    "NULL" text appearing) when writing .catg files; pipeline dialog is
    shorter and the stats sidebar shows compact single-line counts
    (Sub-catchment/Junction/Centroid/Reaches) instead of the print-node list.
v2.7: Build .catg: confluence layer template gains print_node/print_code fields; Print Node Settings panel lets you assign print codes (7/7.1/7.2) interactively and write them back to the layer; stats sidebar shows feature counts after pipeline runs; pyromb now writes print instructions into the control vector at marked confluences. Run RORB: fix ISA table row names in non-lumped mode.
v2.6: Results Viewer: hydrograph viewer highlights the representative TP for the selected duration (not always the critical duration's TP); duration combo label shows actual critical duration (e.g. "Critical (6 hr)") instead of "Critical (auto)".
v2.5: Fix SyntaxError on QGIS 3.28 (Python 3.9) — replace match/case statements in bundled pyromb vendor code with equivalent if/elif chains.
v2.4: Build .catg: error reaches and nodes shown with red thick highlight and red bold ID labels on map canvas (replaces yellow selection).
v2.3: Build .catg: after running checks, error reaches (ID mismatch, unmatched, negative slope) and error nodes (isolated, extra outlets) are automatically selected in the QGIS map so they are visually highlighted.
v2.2: Fix Results Viewer crash (silent segfault) on QGIS 3.38+ when clicking the toolbar icon — defer setFloating(True) to next event loop tick.
v2.1: Run RORB: AEP labels normalised ('63.20%' matches '63.2%'); extreme AEPs ('1 in 10000', '1 in 3684613'/PMP etc.) mapped to rare temporal patterns automatically; AEP combo filtered to recognisable labels; duration combo shows only durations present in both IFD and temporal-patterns CSVs.
v2.0: Build .catg: named shapefiles added to layer panel when saved to folder; single-outlet topology check added. Run RORB: batch-from-.stm-folder mode; Build .catg embedded as first tab. Results Viewer: RORB <v6.52 time-axis correction flagged in Files tab, Hydrograph Viewer preview, and Export warning banner; export message shows actual time step per file.
v1.16: Lumped ISA row count read from .catg control vector (all -99 = 1 group); row count no longer hardcoded.
v1.15: Lumped mode now shows 1 row (Num ISA :1) instead of one row per sub-area.
v1.14: Fix ISA label overflow in .par writer — 10+ catchments no longer trigger Fortran decimal error.
v1.12: Fix ModuleNotFoundError for pyromb on initGui() — each file that uses
    pyromb now has a self-contained fallback to the bundled vendor copy,
    so the plugin loads even if pyromb is not pip-installed.
v1.11: Remove "Run RORB" button — suite now contains only the catchment tools
    and results viewer (3 icons).
v1.1: Add "Run RORB" toolbar icon and bundle pyromb.
v1.0: Initial combined release — RORB Catchment Tools + RORB Results Viewer in one toolbar.
