about=A combined RORB toolbox with three toolbar icons:
    1. Create RORB Layers — create centroid, confluence and reach layer templates
    2. Build RORB .catg — auto-name layers, check links, build RORB/URBS control vectors
    3. RORB Results Viewer — browse .out files, plot hydrographs, export critical events

    Bundles pyromb>=0.3 for the build tools — no separate install needed.

changelog=
v2.5: Fix SyntaxError on QGIS 3.28 (Python 3.9) — replace match/case statements in bundled pyromb vendor code with equivalent if/elif chains.
v2.4: Build .catg: error reaches and nodes shown with red thick highlight and red bold ID labels on map canvas (replaces yellow selection).
v2.3: Build .catg: after running checks, error reaches (ID mismatch, unmatched, negative slope) and error nodes (isolated, extra outlets) are automatically selected in the QGIS map so they are visually highlighted.
v2.2: Fix Results Viewer crash (silent segfault) on QGIS 3.38+ when clicking the toolbar icon — defer setFloating(True) to next event loop tick.
v2.1: Run RORB: AEP labels normalised ('63.20%' matches '63.2%'); extreme AEPs ('1 in 10000', '1 in 3684613'/PMP etc.) mapped to rare temporal patterns automatically; AEP combo filtered to recognisable labels; duration combo shows only durations present in both IFD and temporal-patterns CSVs.
v2.0: Build .catg: named shapefiles added to layer panel when saved to folder; single-outlet topology check added. Run RORB: batch-from-.stm-folder mode; Build .catg embedded as first tab. Results Viewer: RORB <v6.52 time-axis correction flagged in Files tab, Hydrograph Viewer preview, and Export warning banner; export message shows actual time step per file.
v1.16: Lumped ISA row count read from .catg control vector (all -99 = 1 group); row count no longer hardcoded.
v1.15: Lumped mode now shows 1 row (Num ISA :1) instead of one row per sub-area.
v1.14: Fix ISA label overflow in .par writer — 10+ catchments no longer trigger Fortran decimal error.
v1.12: Fix ModuleNotFoundError for pyromb on initGui() — each file that uses
    pyromb now has a self-contained fallback to the bundled vendor copy,
    so the plugin loads even if pyromb is not pip-installed.
v1.11: Remove "Run RORB" button — suite now contains only the catchment tools
    and results viewer (3 icons).
v1.1: Add "Run RORB" toolbar icon and bundle pyromb.
v1.0: Initial combined release — RORB Catchment Tools + RORB Results Viewer in one toolbar.
