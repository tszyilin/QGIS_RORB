about=A combined RORB toolbox with four toolbar icons:
    1. Create RORB Layers — create centroid, confluence and reach layer templates
    2. Build RORB .catg — auto-name layers, check links, build RORB/URBS control vectors
    3. Run RORB — run RORB_CMD.exe against a .catg/.stm pair with per-area kc/m/IL/CL
    4. RORB Results Viewer — browse .out files, plot hydrographs, export critical events

    Bundles pyromb>=0.3 for the build tools — no separate install needed.
    Running RORB requires a separately licensed RORBwin installation
    (RORB_CMD.exe).
