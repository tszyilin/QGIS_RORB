about=RORB Catchment Tools: A combined plugin for preparing and building RORB and URBS
    runoff model control vectors from catchment GIS layers.

    Preparation tools:
    - Auto-name subcatchment polygons south to north (numeric IDs)
    - Auto-name centroid points south to north (letter IDs)
    - Auto-name confluence points south to north (lowercase letter IDs)
    - Auto-name reaches from connected node IDs
    - Check RORB layer link consistency
    - Clean up negative slope values in reach layers

    Build tools:
    - Build RORB control vector (.cat file)
    - Build URBS control vector (.vec + .cat files)

    Bundles the pyromb (>=0.3) library for the build tools — no separate
    install needed.
